## Usage

1. Write down script using ProcGen language and primitives defined in `svg.procgen`. 
2. Run `./svgprocgen <scriptName> <pictureName.svg>` to get SVG picture.
3. Use *convert* from *ImageTricks* to convert SVG to PNG.
